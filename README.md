# How to Say This in Another Way

A web-based language assistant for bilingual and second-language learners who want better ways to express words or short phrases.

## What it does

- Lets users enter a word or short phrase
- Fetches alternative expressions from the Datamuse API
- Groups suggestions into tone categories: formal, casual, and academic
- Shows example sentences for each suggestion
- Lets users save phrases for later review
- Stores saved phrases in localStorage so they persist in the browser

## Tech stack

- React
- Vite
- JavaScript
- CSS
- Datamuse API
- localStorage

## How to run locally

1. Make sure Node.js is installed
2. Clone the repository
3. Open the project folder in a terminal
4. Run:

```bash

npm run dev
